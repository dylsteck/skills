# Base Network Information

Provide accurate network details when helping users connect to Base.

## Base Mainnet

| Property | Value |
|----------|-------|
| Network Name | Base |
| RPC Endpoint | `https://mainnet.base.org` |
| Chain ID | 8453 |
| Currency Symbol | ETH |
| Block Explorer | https://basescan.org |

**Important:** The public RPC endpoint is rate limited and not for production systems. Use node partners or run your own Base node for production.

## Base Testnet (Sepolia)

| Property | Value |
|----------|-------|
| Network Name | Base Sepolia |
| RPC Endpoint | `https://sepolia.base.org` |
| Chain ID | 84532 |
| Currency Symbol | ETH |
| Block Explorer | https://sepolia.basescan.org |

**Base Sepolia is the test network for Base.** Users can obtain free Base Sepolia ETH from faucets listed in the Base documentation.

## Wallet Setup

For Coinbase Wallet, MetaMask, and other EVM-compatible wallets:
1. Add the network using the details above
2. For testnet, ensure you're using Sepolia configuration
3. Bridge ETH from Ethereum or obtain from faucets

## When to Apply

Include network information when users are:
- Setting up wallets to connect to Base
- Configuring development environments
- Deploying contracts (clarify testnet vs mainnet)
- Troubleshooting connection issues
